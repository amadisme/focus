# 智能防久坐应用 - 用户使用指南

## 📦 安装说明

### macOS 用户

#### 方式1：安装 DMG（推荐）
1. 下载 `智能防久坐-1.0.0-x64.dmg`（Intel芯片）或 `智能防久坐-1.0.0-arm64.dmg`（M系列芯片）
2. 双击打开 DMG 文件
3. 将应用图标拖到 Applications 文件夹
4. 在 Launchpad 或 Applications 文件夹中找到并启动应用

#### 方式2：使用 ZIP
1. 下载 `智能防久坐-1.0.0-x64.zip` 或 `智能防久坐-1.0.0-arm64.zip`
2. 解压 ZIP 文件
3. 将应用移动到 Applications 文件夹
4. 启动应用

**首次启动注意**：
- macOS 可能提示"无法打开，因为它来自身份不明的开发者"
- 解决方法：右键点击应用 → 选择"打开" → 点击"打开"确认
- 或者：系统偏好设置 → 安全性与隐私 → 点击"仍要打开"

### Windows 用户

#### 方式1：安装版（推荐）
1. 下载 `智能防久坐-1.0.0-x64.exe`
2. 双击运行安装程序
3. 选择安装位置（或使用默认）
4. 完成安装后，从开始菜单或桌面快捷方式启动

#### 方式2：便携版
1. 下载 `智能防久坐-1.0.0-x64.exe`（便携版）
2. 将文件放到任意文件夹
3. 双击运行（无需安装）

**首次启动注意**：
- Windows 可能显示 SmartScreen 警告
- 点击"更多信息" → 点击"仍要运行"

## 🚀 快速开始

### 1. 首次启动
- 应用会出现在系统托盘（macOS右上角 / Windows右下角）
- 同时显示一个悬浮窗，展示当前工作时间

### 2. 主窗口
- **时间显示**：当前累计工作时间（分钟:秒）
- **状态显示**：专注中 / 休息中 / 该休息啦
- **进度条**：距离目标时间的进度
- **设置按钮**：右上角 ⚙️ 图标
- **关闭按钮**：右上角 ✕ 图标（关闭窗口，应用继续运行）

### 3. 系统托盘
右键点击托盘图标，可以：
- 查看当前工作时间和状态
- 重置计时器
- 打开设置
- 显示主窗口
- 退出应用

## ⚙️ 配置设置

### 打开设置界面
- 点击主窗口右上角 ⚙️ 按钮
- 或右键托盘图标 → 选择"⚙️ 设置"

### 可配置参数

#### 1. 目标工作时间（默认：45分钟）
- 连续工作达到此时间后，会提醒你休息
- 推荐：25分钟（番茄工作法）/ 45分钟（标准）/ 90分钟（深度专注）

#### 2. 离座重置阈值（默认：10分钟）
- 离开电脑超过此时间，计时器会重置为0
- 推荐：10分钟（平衡）/ 15分钟（宽松）

#### 3. 强提醒时间（默认：50分钟）
- 工作超过此时间，会发送更强烈的提醒
- 必须大于"目标工作时间"

#### 4. 活跃判定阈值（默认：5秒）
- 停止操作小于此时间，视为活跃工作
- 推荐：5秒（平衡）/ 10秒（宽松，适合阅读）

### 保存设置
1. 修改参数后，点击"保存设置"
2. 会提示"设置已保存！请重启应用使设置生效"
3. 右键托盘图标 → 退出
4. 重新启动应用

## 💡 工作原理

### 智能三态识别

应用会根据你的电脑使用情况，自动识别三种状态：

#### 🟢 活跃工作（闲置 < 5秒）
- 正在打字、移动鼠标
- **计时器累加**

#### 🟡 暂停状态（5秒 ~ 10分钟）
- 阅读文档、思考、短暂休息（喝水、上厕所）
- **计时器暂停**（不累加，也不重置）
- 保留你的工作成果

#### 🔴 离座状态（闲置 ≥ 10分钟）
- 开会、午休、外出
- **计时器重置为0**
- 说明你已经休息过了

### 实际场景示例

```
场景1：短暂休息
工作30分钟 → 喝水5分钟 → 继续工作10分钟
结果：30分钟 → 暂停 → 40分钟 ✅
说明：保留工作成果，不会因为喝水就清零

场景2：真正休息
工作40分钟 → 开会15分钟 → 继续工作
结果：40分钟 → 重置为0 → 重新计时 ✅
说明：开会已达到休息目的，重新计时
```

## 🔔 提醒功能

### 第一次提醒（达到目标时间）
- 标题：⏰ 该休息啦！
- 内容：您已连续专注工作 XX 分钟，建议稍作休息。
- 状态：主窗口显示黄色"⏰ 该休息啦！"

### 第二次提醒（超过强提醒时间）
- 标题：🚨 强烈建议休息！
- 内容：您已连续工作 XX 分钟，长时间久坐对健康不利，请务必休息！
- 状态：主窗口显示红色"🚨 强烈建议休息！"并闪烁

## 🛠️ 常见问题

### Q1: 应用图标在哪里？
**A**: 
- macOS：屏幕右上角菜单栏（系统托盘）
- Windows：屏幕右下角任务栏（系统托盘）

### Q2: 如何完全退出应用？
**A**: 
- 右键点击托盘图标 → 选择"退出"
- 或者：Command+Q (macOS) / Alt+F4 (Windows)

### Q3: 为什么计时器不动了？
**A**: 
- 这是正常的！说明你停止操作超过5秒
- 应用认为你在阅读或思考，所以暂停计时
- 继续操作后会自动恢复

### Q4: 为什么计时器突然归零了？
**A**: 
- 说明你离开电脑超过10分钟
- 应用认为你已经休息过了，所以重置计时
- 这是智能功能，避免开会回来还提醒你休息

### Q5: macOS 提示需要"辅助功能"权限？
**A**: 
1. 打开"系统偏好设置"
2. 进入"安全性与隐私"
3. 选择"隐私" → "辅助功能"
4. 点击左下角🔒解锁
5. 勾选"智能防久坐"应用
6. 重启应用

### Q6: 收不到通知？
**A**: 
- macOS：系统偏好设置 → 通知 → 找到应用并允许通知
- Windows：设置 → 系统 → 通知和操作 → 允许应用通知
- 确认没有开启"勿扰模式"

### Q7: 如何修改提醒时间？
**A**: 
1. 点击主窗口右上角 ⚙️ 按钮
2. 修改"目标工作时间"
3. 点击"保存设置"
4. 重启应用

### Q8: 卸载应用
**A**: 
- macOS：将应用从 Applications 文件夹拖到废纸篓
- Windows：控制面板 → 程序和功能 → 卸载
- 配置文件位置（可手动删除）：
  - macOS: `~/Library/Application Support/smart-anti-sedentary`
  - Windows: `%APPDATA%\smart-anti-sedentary`
