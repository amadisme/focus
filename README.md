# Smart Anti-Sedentary App - 智能防久坐提示应用

## 项目简介

这是一个基于系统闲置时间API的智能防久坐提示应用，与传统的固定定时器不同，本应用能够：

- ✅ **智能识别**：只在真正连续工作时计时
- ✅ **自动重置**：离座超过5分钟自动清零
- ✅ **不打断工作流**：不会在你刚回来时误报
- ✅ **跨平台支持**：支持 Windows、macOS 和 Linux

## 核心特性

### 1. 真正的智能计时逻辑 🧠
本应用的核心创新在于**三态智能判定**：

- **活跃工作**（闲置 < 5秒）：✅ 计时累加
  - 正在打字、移动鼠标、操作电脑
  
- **暂停状态**（5秒 ≤ 闲置 < 10分钟）：⏸️ 计时暂停
  - 阅读文档、思考、短暂休息（喝水、上厕所）
  - **不累加，但也不重置**（保留工作成果）
  
- **离座状态**（闲置 ≥ 10分钟）：🔄 计时重置
  - 开会、午休、真正离开工位
  - 完全重置为0，重新开始计时

**为什么是10分钟？**
- 短暂休息（喝水5分钟）→ 暂停计时，回来继续
- 真正离开（开会30分钟）→ 重置计时，已达到休息目的

这样就不会出现：
- ❌ 刚开会回来就提醒（传统定时器的问题）
- ❌ 喝水5分钟回来计时器归零（过于敏感）
- ✅ 只在真正连续工作时才提醒休息！

### 2. 完全可配置 ⚙️
**无需修改代码，通过UI界面配置所有参数！**

- 目标工作时间：45分钟（可设置 1-120 分钟）
- 离座重置阈值：10分钟（可设置 1-60 分钟）
- 强提醒时间：50分钟（可设置 1-120 分钟）
- 活跃判定阈值：5秒（可设置 1-30 秒）

**如何配置**：
- 点击主窗口右上角 ⚙️ 按钮
- 或右键点击托盘图标 → 设置
- 修改参数后保存，重启生效

详见 [配置指南](CONFIG_GUIDE.md)

### 3. 两级提醒系统
- **Phase 1**（45分钟）：温和提醒该休息
- **Phase 2**（50分钟）：强烈建议休息

### 4. 友好的用户界面
- 悬浮窗显示当前工作时间和状态
- 系统托盘图标实时更新
- 系统通知提醒

- **Node.js**: 后端运行时
- **HTML/CSS/JavaScript**: 前端界面

## 工作原理

1. 应用每秒轮询一次系统闲置时间
2. **活跃工作**（闲置 < 5秒）：累加工作时间
3. **暂停状态**（5秒 ≤ 闲置 < 10分钟）：计时暂停，保留当前时间
4. **离座状态**（闲置 ≥ 10分钟）：计时重置为0
5. 当累计工作时间达到45分钟时，发送第一次提醒
6. 当累计工作时间达到50分钟时，发送强提醒

### 实际场景示例
```
场景1：短暂休息（喝水5分钟）
工作30分钟 → 离开5分钟 → 回来继续工作
结果：30分钟 → 暂停 → 继续累加 ✅
说明：保留工作成果，不会因为喝水就清零

场景2：真正休息（开会15分钟）
工作40分钟 → 开会15分钟 → 回来继续工作
结果：40分钟 → 重置为0 → 重新开始 ✅
说明：已达到休息目的，重新计时
```

## 配置说明

### 推荐方式：使用设置界面 ⚙️
1. 打开应用后，点击右上角设置按钮
2. 调整所需参数
3. 保存并重启应用

### 高级方式：直接编辑配置文件
配置文件位置：
- macOS: `~/Library/Application Support/smart-anti-sedentary/config/settings.json`
- Windows: `%APPDATA%\smart-anti-sedentary\config\settings.json`
- Linux: `~/.config/smart-anti-sedentary/config/settings.json`

详细说明请查看 [配置指南](CONFIG_GUIDE.md)
