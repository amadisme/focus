### 4. 计时功能

**测试步骤**:
1. 启动应用
2. 开始工作（移动鼠标/打字）
3. 观察时间是否累加

**预期结果**:
- [ ] 时间每秒增加 1
- [ ] 进度条随时间增长
- [ ] 托盘菜单的时间同步更新

---

### 5. 自动重置功能

**测试步骤**:
1. 启动应用，工作一段时间（如1分钟）
2. 停止移动鼠标/键盘超过5分钟
3. 观察计时器是否重置

**预期结果**:
- [ ] 离开电脑5分钟后，时间自动归零
- [ ] 状态变为 "休息中"
- [ ] 控制台显示 "检测到用户离座/开会..."
- [ ] 回来后重新开始计时

⚠️ **提示**: 为了快速测试，可以临时修改 `RESET_THRESHOLD` 为 `10`（10秒）：
```javascript
// src/main.js
const RESET_THRESHOLD = 10; // 临时改为10秒测试
```

### 6. 提醒功能

#### Phase 1 提醒（45分钟）

**测试步骤**:
1. 临时修改 `TARGET_WORK_TIME` 为 `10`（10秒）
   ```javascript
   // src/main.js
   const TARGET_WORK_TIME = 10;
   ```
2. 重启应用
3. 工作10秒

**预期结果**:
- [ ] 10秒后显示系统通知
- [ ] 通知标题: "⏰ 该休息啦！"
- [ ] 通知内容: "您已连续专注工作 0 分钟，建议稍作休息。"
- [ ] 主窗口状态变为 "⏰ 该休息啦！"（黄色）
- [ ] 进度条达到 100%

#### Phase 2 提醒（50分钟）

**测试步骤**:
1. 继续上述测试
2. 临时修改 `REMIND_PHASE_2_TIME` 为 `15`（15秒）
3. 再工作5秒

**预期结果**:
- [ ] 15秒后显示强提醒通知
- [ ] 通知标题: "🚨 强烈建议休息！"
- [ ] 主窗口状态变为 "🚨 强烈建议休息！"（红色，脉冲动画）
- [ ] 进度条变为红色渐变

### 7. 重置功能

**测试步骤**:
1. 工作一段时间（如30秒）
2. 点击主窗口的 "重置计时器" 按钮

**预期结果**:
- [ ] 时间立即归零
- [ ] 进度条回到 0%
- [ ] 状态恢复为 "专注中..."
- [ ] 重新开始计时

**测试步骤2**:
1. 工作一段时间
2. 右键点击托盘图标
3. 选择 "重置计时器"

**预期结果**:
- [ ] 效果与点击窗口按钮相同
