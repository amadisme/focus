## 👥 用户如何安装

### macOS 用户安装步骤

1. **下载文件**
   - M1/M2/M3 用户下载 `arm64` 版本
   - Intel 用户下载 `x64` 版本

2. **安装应用**
   - 双击打开 DMG 文件
   - 拖动应用到 Applications 文件夹
   - 完成！

3. **首次打开**
   - 在 Launchpad 或 Applications 文件夹找到应用
   - 双击打开
   - 如果提示"无法打开"：
     - 右键点击应用 → 选择"打开" → 点击"打开"

4. **开始使用**
   - 应用会出现在系统托盘（右上角菜单栏）
   - 同时显示一个悬浮窗
   - 右键托盘图标可以打开设置

## 🌐 推荐的分享方式

### 方式1：网盘分享（最简单）

1. **上传到网盘**
   ```
   上传这些文件：
   - 智能防久坐-1.0.0-arm64.dmg
   - 智能防久坐-1.0.0-x64.dmg

   ```

2. **创建分享链接**
   - 在网盘中右键选择"分享"
   - 复制分享链接
   - 发送给需要的人

3. **告诉用户**
   ```
   这是智能防久坐应用：
   
   下载地址：[网盘链接]
   
   M1/M2/M3 芯片 Mac：下载 arm64 版本
   Intel 芯片 Mac：下载 x64 版本
   
   使用说明在压缩包里的 USER_GUIDE.md
   ```

### 方式2：GitHub Releases（推荐，免费）

1. **创建 GitHub 仓库**
   ```bash
   # 如果还没有
   git init
   git add .
   git commit -m "智能防久坐应用 v1.0.0"
   git remote add origin https://github.com/你的用户名/smart-anti-sedentary.git
   git push -u origin main
   ```

2. **创建 Release**
   - 访问你的 GitHub 仓库
   - 点击 "Releases" → "Create a new release"
   - 填写：
     - Tag: `v1.0.0`
     - Title: `智能防久坐 v1.0.0`
     - Description: 从 USER_GUIDE.md 复制简介
   - 上传 dist 文件夹中的所有 DMG 和 ZIP 文件
   - 上传 USER_GUIDE.md
   - 点击 "Publish release"

3. **分享下载链接**
   ```
   https://github.com/你的用户名/smart-anti-sedentary/releases/latest
   ```

### 方式3：直接发送（小范围）

如果只是给几个朋友：
- 微信/QQ：直接发送文件（可能需要压缩）
- AirDrop：Mac 之间直接传送

## ⚠️ 常见问题

### Q: 为什么没有 Windows 版本？

A: 需要在 Windows 电脑上打包，或者运行：
```bash
npm run build:win
```
（在 macOS 上打包 Windows 版本可能失败）

### Q: 文件太大了（90MB+）

A: 这是正常的！因为 Electron 应用包含了：
- 完整的 Chromium 浏览器引擎
- Node.js 运行时
- 你的应用代码

可以压缩后分享，用户解压即可。

### Q: 如何更新应用？

A: 
1. 修改代码后，更新 `package.json` 中的版本号
2. 重新运行 `npm run build:mac`
3. 分享新版本的文件
