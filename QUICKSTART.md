# 快速开始指南

## 1. 测试系统闲置时间检测

在运行完整应用之前，可以先测试闲置时间检测是否正常工作：

```bash
npm test
```

你应该能看到：
- 显示 Electron 和 Node.js 版本
- 每2秒显示一次当前闲置时间
- 当你移动鼠标或按键时，闲置时间会重置为接近0
- 测试运行5次后自动退出

⚠️ **注意**: 不要使用 `node test-idle.js`，因为 desktop-idle 是为 Electron 编译的原生模块，必须在 Electron 环境中运行。

## 2. 启动应用

```bash
npm start
```

## 3. 使用说明

### 主窗口
- 启动后会显示一个悬浮窗，显示当前工作时间和状态
- 可以拖动窗口到任意位置
- 点击右上角的 ✕ 可以关闭主窗口（应用继续在后台运行）

### 系统托盘
- 应用会在系统托盘显示图标
- 点击图标可以打开/关闭主窗口
- 右键点击图标可以：
  - 查看当前工作时间和状态
  - 重置计时器
  - 退出应用

### 工作原理
1. 当你开始工作时，计时器开始累加
2. 如果你离开电脑超过5分钟，计时器会自动清零
3. 工作45分钟后，会收到第一次温和提醒
4. 工作50分钟后，会收到强提醒
5. 起身休息5分钟后，计时器会自动重置，重新开始计时

### 提示
- 这个应用不会在你刚开完会回来时误报
- 它只在你真正连续工作时才计时
- 适合需要长时间专注工作的人使用

## 4. 自定义配置

如果想修改工作时间或重置阈值，编辑 `src/main.js`：

```javascript
// 在文件顶部找到这些常量
const TARGET_WORK_TIME = 45 * 60;      // 改为你想要的分钟数 * 60
const RESET_THRESHOLD = 5 * 60;        // 改为你想要的离座阈值 * 60
const REMIND_PHASE_2_TIME = 50 * 60;   // 改为你想要的强提醒时间 * 60
```

## 5. 故障排除

### 如果闲置时间检测不工作
- 确保你的操作系统权限设置正确
- macOS 用户可能需要在"系统偏好设置 > 安全性与隐私 > 辅助功能"中授权
- Windows 用户可能需要以管理员身份运行

### 如果应用无法启动
- 检查 Node.js 版本（建议 16.x 或更高）
- 重新安装依赖：`rm -rf node_modules && npm install`
- 查看控制台错误信息

## 6. 开发模式

如果你想修改代码并调试：

```bash
npm run dev
```

这会启动应用并开启 Chrome DevTools 调试端口。
