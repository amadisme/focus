# ⚙️ 配置化功能使用指南

## 功能概述

现在应用支持完全可配置的时间参数，无需修改代码即可调整：
- ✅ 目标工作时间（提醒阈值）
- ✅ 离座重置阈值
- ✅ 强提醒时间
- ✅ 活跃判定阈值

所有配置都会自动保存，下次启动时加载。

---

## 打开设置界面

### 方法1：通过主窗口
1. 启动应用：`npm start`
2. 点击主窗口右上角的 ⚙️ 设置按钮

### 方法2：通过系统托盘
1. 右键点击系统托盘图标
2. 选择 "⚙️ 设置"

## 配置参数说明

### 1. 目标工作时间
**默认值**: 45 分钟  
**范围**: 1-120 分钟  
**说明**: 连续工作达到此时间后，发送第一次提醒

**使用建议**：
- 25 分钟：番茄工作法
- 45 分钟：标准办公（推荐）
- 90 分钟：深度专注

### 2. 离座重置阈值
**默认值**: 10 分钟  
**范围**: 1-60 分钟  
**说明**: 闲置超过此时间，计时器重置为0

**使用建议**：
- 5 分钟：对短暂休息敏感（不推荐）
- 10 分钟：平衡方案（推荐）
- 15 分钟：对休息宽容

**实际场景**：
```
5分钟阈值：
- 喝水5分钟 → 重置 ❌
- 上厕所 → 重置 ❌

10分钟阈值（推荐）：
- 喝水5分钟 → 暂停 ✅
- 开会15分钟 → 重置 ✅

15分钟阈值：
- 喝水5分钟 → 暂停 ✅
- 短会10分钟 → 暂停 ❌
```

### 3. 强提醒时间
**默认值**: 50 分钟  
**范围**: 1-120 分钟  
**说明**: 连续工作超过此时间，发送强烈提醒

**注意**: 必须大于"目标工作时间"

**使用建议**：
- 目标45分钟 → 强提醒50分钟
- 目标25分钟 → 强提醒30分钟
- 目标90分钟 → 强提醒100分钟

### 4. 活跃判定阈值
**默认值**: 5 秒  
**范围**: 1-30 秒  
**说明**: 闲置小于此时间视为活跃工作，计时累加

**使用建议**：
- 3 秒：敏感，适合快节奏工作
- 5 秒：平衡（推荐）
- 10 秒：宽松，阅读友好

**实际影响**：
```
3秒阈值：
- 停顿思考5秒 → 计时暂停

5秒阈值（推荐）：
- 停顿思考5秒 → 计时暂停
- 持续打字 → 计时累加

10秒阈值：
- 停顿思考8秒 → 计时继续
- 阅读10秒 → 计时继续
```

## 操作步骤

### 修改配置
1. 打开设置窗口
2. 修改需要调整的参数
3. 点击"保存设置"按钮
4. 看到提示："设置已保存！请重启应用使设置生效。"
5. 退出应用
6. 重新启动：`npm start`
7. 新配置生效 ✅

### 恢复默认
1. 打开设置窗口
2. 点击"恢复默认"按钮
3. 确认操作
4. 点击"保存设置"
5. 重启应用

## 配置文件位置

配置自动保存在：
```
macOS:
~/Library/Application Support/smart-anti-sedentary/config/settings.json

Windows:
%APPDATA%\smart-anti-sedentary\config\settings.json

Linux:
~/.config/smart-anti-sedentary/config/settings.json
```

### 配置文件格式
```json
{
  "targetWorkTime": 45,
  "resetThreshold": 10,
  "remindPhase2Time": 50,
  "activeThreshold": 5
}
```

### 手动编辑（高级）
你也可以直接编辑配置文件：
1. 退出应用
2. 编辑 `settings.json`
3. 重新启动应用

**注意**: 手动编辑时请确保：
- JSON 格式正确
- 数值在有效范围内
- 强提醒时间 > 目标工作时间

## 故障排除

### 问题1: 设置窗口无法打开
**解决方案**:
- 检查文件是否存在：`src/renderer/settings.html`
- 重新安装：`npm install`

### 问题2: 修改后没有生效
**原因**: 需要重启应用
**解决方案**:
1. 完全退出应用（托盘菜单 → 退出）
2. 重新运行 `npm start`

### 问题3: 配置丢失
**原因**: 配置文件损坏
**解决方案**:
1. 删除配置文件（见上方路径）
2. 重启应用（会自动创建默认配置）

### 问题4: 保存失败
**可能原因**:
- 磁盘空间不足
- 权限问题
- JSON 格式错误

**解决方案**:
1. 检查磁盘空间
2. 确保应用有写入权限
3. 使用"恢复默认"功能

## UI 窗口调整

### 主窗口尺寸
现在已调整为 350 x 280 像素，确保所有内容可见，无需滚动。

如需进一步调整，编辑 `src/main.js`：
```javascript
mainWindow = new BrowserWindow({
  width: 350,
  height: 280,  // 调整这个值
  // ...
});
```

### 设置窗口尺寸
默认 420 x 550 像素，如需调整：
```javascript
settingsWindow = new BrowserWindow({
  width: 420,
  height: 550,  // 调整这个值
  // ...
});
```
