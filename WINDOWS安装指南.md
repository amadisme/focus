# 💻 Windows 11 用户安装指南

## 🚀 安装方法

### 方式1：安装版（推荐）

#### 步骤：

1. **下载安装包**
   - 下载 `智能防久坐 Setup 1.0.0.exe`

2. **运行安装程序**
   - 双击下载的 EXE 文件
   
3. **处理 Windows 安全警告**
   - Windows 可能显示 "Windows 已保护你的电脑"
   - 点击 **"更多信息"**
   - 点击 **"仍要运行"**

4. **安装向导**
   - 选择安装位置（或使用默认）
   - 可以勾选：
     - ✅ 创建桌面快捷方式
     - ✅ 创建开始菜单快捷方式
   - 点击 **"安装"**

5. **完成安装**
   - 点击 **"完成"**
   - 勾选 "启动应用" 可以立即运行

6. **启动应用**
   - 从桌面快捷方式启动，或
   - 从开始菜单搜索 "智能防久坐"

---

### 方式2：便携版（无需安装）

#### 步骤：

1. **下载便携版**
   - 下载 `智能防久坐 1.0.0.exe`

2. **放到合适的位置**
   - 创建文件夹，如 `C:\Programs\智能防久坐\`
   - 将 EXE 文件放进去

3. **处理安全警告**
   - 双击运行 EXE 文件
   - 如果出现警告，点击 "更多信息" → "仍要运行"

4. **直接使用**
   - 无需安装，直接运行
   - 可以创建桌面快捷方式

## 🔍 启动后在哪里找到应用？

### 1. 系统托盘（任务栏右下角）
- 应用启动后，会在**系统托盘**显示图标
- 位置：屏幕右下角，时间旁边
- 如果没看到，点击 **^** (向上箭头) 展开隐藏图标

### 2. 悬浮窗
- 启动后会显示一个小窗口
- 显示当前工作时间
- 可以拖动到屏幕任意位置
- 点击右上角 ✕ 可以关闭（应用继续运行）

## 🎮 如何使用

### 基本操作

#### 右键托盘图标：
```
工作时间: 00:00          ← 当前累计时间
状态: 专注中              ← 当前状态
─────────────────────
重置计时器               ← 重新开始计时
显示主窗口               ← 打开悬浮窗
─────────────────────
⚙️ 设置                 ← 配置参数
─────────────────────
退出                     ← 完全退出应用
```

#### 主窗口功能：
- **时间显示**：当前工作时长（分钟:秒）
- **状态显示**：专注中 / 休息中 / 该休息啦
- **进度条**：距离目标时间的进度
- **⚙️ 按钮**：打开设置界面
- **✕ 按钮**：关闭窗口（应用继续运行）

## ⚙️ 配置设置

### 打开设置：
1. 右键托盘图标 → 点击 "⚙️ 设置"
2. 或点击主窗口右上角 ⚙️ 按钮

### 可以修改的参数：

| 参数 | 默认值 | 说明 |
|-----|-------|------|
| 目标工作时间 | 45分钟 | 工作多久后提醒休息 |
| 离座重置阈值 | 10分钟 | 离开多久后重置计时 |
| 强提醒时间 | 50分钟 | 超过多久发送强烈提醒 |
| 活跃判定阈值 | 5秒 | 停止操作多久算暂停 |

### 修改步骤：
1. 在设置界面输入新的数值
2. 点击 **"保存设置"**
3. 退出应用（右键托盘 → 退出）
4. 重新启动应用

## 💡 工作原理

### 智能识别三种状态：

#### 🟢 活跃工作（闲置 < 5秒）
- 正在打字、移动鼠标
- **计时器累加** ⏱️

#### 🟡 暂停状态（5秒 ~ 10分钟）
- 阅读文档、思考问题、短暂休息
- **计时器暂停** ⏸️（保留当前时间）

#### 🔴 离座状态（闲置 ≥ 10分钟）
- 开会、午休、外出
- **计时器重置为 0** 🔄（已休息过了）

### 实际场景示例：

```
场景1：短暂休息
工作30分钟 → 喝水5分钟 → 继续工作
结果：30分钟 → 暂停 → 40分钟 ✅

场景2：长时间离开
工作40分钟 → 开会15分钟 → 继续工作  
结果：40分钟 → 重置为0 → 重新计时 ✅
```

## 🔔 提醒通知

### 第一次提醒（45分钟）
- 弹出通知："⏰ 该休息啦！"
- 建议稍作休息

### 第二次提醒（50分钟）
- 弹出通知："🚨 强烈建议休息！"
- 长时间久坐对健康不利

### 如果没有收到通知：
1. 检查 Windows 通知设置
2. 设置 → 系统 → 通知
3. 确保允许应用通知
4. 关闭"勿扰模式"

## ❓ 常见问题

### Q1: 应用在哪里？
**A**: 系统托盘（屏幕右下角，时间旁边）
- 如果没看到，点击向上箭头 ^

### Q2: 如何完全退出？
**A**: 右键托盘图标 → 点击 "退出"

### Q3: 为什么计时器不动了？
**A**: 这是正常的！
- 说明你停止操作超过5秒
- 应用认为你在阅读或思考
- 继续操作后会自动恢复

### Q4: 为什么计时器突然归零？
**A**: 说明你离开电脑超过10分钟
- 应用认为你已经休息过了
- 避免开会回来还提醒你休息

### Q5: 开机自启动？
**A**: 
1. 按 Win + R
2. 输入 `shell:startup`
3. 将应用快捷方式拖进去

### Q6: 卸载应用
**A**: 
- 设置 → 应用 → 已安装的应用
- 找到 "智能防久坐" → 卸载
- 或：控制面板 → 程序和功能 → 卸载

### Q7: 配置文件在哪里？
**A**: `%APPDATA%\smart-anti-sedentary\config\settings.json`
- 卸载应用时可以手动删除
