# 应用截图和演示

## 主界面预览

应用主界面包含以下元素：

```
┌─────────────────────────────────┐
│  💻 专注计时器              ✕  │
│                                  │
│          00:00                   │
│        专注中...                 │
│                                  │
│  ▓▓▓▓▓▓░░░░░░░░░░░░░░░░░░░░░   │
│                                  │
│  目标时间: 45分钟                │
│  离座重置: 5分钟                 │
│                                  │
│     [ 重置计时器 ]               │
└─────────────────────────────────┘
```

## 界面特点

### 1. 标题栏
- 显示 "💻 专注计时器"
- 右侧有关闭按钮 ✕
- 可以拖动移动窗口

### 2. 计时显示
- 大号数字显示当前工作时间（MM:SS格式）
- 下方显示当前状态文字

### 3. 进度条
- 根据目标时间动态显示进度
- 未到45分钟：蓝色渐变
- 超过45分钟：红色渐变
- 平滑动画效果

### 4. 信息区域
- 显示目标工作时间
- 显示离座重置阈值

### 5. 操作按钮
- "重置计时器" 按钮
- 悬停效果
- 点击反馈

## 颜色主题

### 背景渐变
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```
- 从蓝紫色 (#667eea) 到深紫色 (#764ba2)
- 135度斜向渐变

### 进度条（正常）
```css
background: linear-gradient(90deg, #56CCF2, #2F80ED);
```
- 从浅蓝色到深蓝色

### 进度条（超时）
```css
background: linear-gradient(90deg, #FF6B6B, #EE5A6F);
```
- 从红色到深红色

## 状态指示

### 工作状态颜色
- **专注中** (working): 白色文字
- **休息中** (idle): 白色文字
- **该休息啦** (phase1): 黄色文字 (#FFD93D) + warning样式
- **强烈建议休息** (phase2): 红色文字 (#FF6B6B) + 脉冲动画

### 脉冲动画
```css
@keyframes pulse {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.6; }
}
```
- 1.5秒循环
- 透明度在 60%-100% 之间变化

## 交互效果

### 按钮悬停
- 背景透明度从 20% 增加到 30%
- 向上移动 2px
- 过渡时间 0.3秒

### 按钮点击
- 回到原始位置
- 立即响应

### 窗口拖动
- 可以拖动整个窗口区域（除了按钮）
- 使用 `-webkit-app-region: drag`
- 按钮区域使用 `-webkit-app-region: no-drag`

## 响应式设计

- 窗口固定大小：350x200 像素
- 不可调整大小
- 始终置顶（alwaysOnTop）
- 透明背景
- 圆角边框（15px）
- 阴影效果（10px blur, 40px spread）

## 字体

```css
font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 
             Roboto, Oxygen, Ubuntu, sans-serif;
```
- 使用系统默认字体
- 跨平台一致性

## 动画细节

### 进度条过渡
```css
transition: width 0.5s ease;
```
- 0.5秒平滑过渡
- ease 缓动函数

### 按钮过渡
```css
transition: all 0.3s;
```
- 0.3秒过渡
- 包括背景、transform等所有属性

## 如何截图

### macOS
1. 启动应用：`npm start`
2. 按 Cmd + Shift + 4 + Space
3. 点击应用窗口

### Windows
1. 启动应用：`npm start`
2. 按 Win + Shift + S
3. 选择窗口区域

### 使用 Electron 截图
```javascript
// 在 renderer.js 中添加
const screenshot = await mainWindow.capturePage();
const fs = require('fs');
fs.writeFileSync('screenshot.png', screenshot.toPNG());
```
